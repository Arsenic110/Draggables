Easy Start

Start by linking the script with <script src="https://cdn.statically.io/gh/Arsenic110/Draggables/main/draggables.js"></script>
You can use whichever CDN you like, here I use statically.io!

Then, you need to call the makeDraggable(HTMLCollection) function on the collection of elements you wish to make draggable. These are referred as 'draggables'.

This can be easily achieved with something like:
<script>makeDraggable(document.getElementsByClassName("drag"))</script>

Configuration

The library makes use of the window.drag JSON object for configuration.
For simplicity, these values are preset by default, but you can change them at simply by changing the values at runtime, after the makeDraggable(HTMLCollection) function has been ran.

{

  • zTop:444 - default zIndex all draggables are elevated to.
  • offset:15 - mouse grab offset, it is resolution-specific, not sure how to calculate this :p
  • momentum - nested JSON object used for the physics behavior.
    • {
    • speed:1 - multiplier to velocity, natural motion is 1. Higher = faster.
    • spaceFriction:0.8 - friction the draggable experiences while moving through space. 1 = no friction, 0 = disables physics.
    • wallFriction:0.7 - simulates plastic collision behavior, i.e. applies an instant burst of friction when the draggable hits a wall.
    }
}
